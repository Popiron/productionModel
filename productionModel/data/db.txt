Плита, Сахар, Вода => Карамель {Добавляем к сахару воду. Растапливаем сахар. Получаем Карамель.}
Плита, Карамель, Курица, Соевый соус, Чеснок, Уксус, Кунжут, Соль, Перец => Карамельная курица {Жарим курицу в смеси соевого соуса и карамели с добавлением чеснока, уксуса, кунжута, соли и перца. Получаем Карамельную курицу.}
Плита, Рис, Вода, Соль => Рис отварной {Варим рис с добавлением соли. Получаем Рис отварной.}
Рис отварной, Карамельная курица, Зеленый Лук => Рис с курицей по-азиатски! {Добавляем рис отварной и карамельную курицу. Сверху присыпаем зеленым луком. Получаем Рис с курицей по-азиатски.}
Пшеница => Мука {Путем перетирания пшеницы получаем Муку.}
Рис => Мука {Путем перетирания риса получаем Муку.}
Гречка => Мука {Путем перетирания гречки получаем Муку.}
Мука, Яйцо, Вода => Тесто {Путем смешивания муки, яйца и воды получаем Тесто.}
Тесто => Лапша {С помощью теста можно сформировать Лапшу.}
Плита, Лапша, Вода, Соль => Лапша вареная {Варим лапшу с добавлением соли. Получаем Лапшу вареную.}
Молоко, Холодильник => Сливки {С помощью молока и холодильника можно сделать Сливки.}
Сливки => Сливочное масло {С помощью сливок и механического воздействия на них можно сделать Сливочное масло.}
Плита, Сливочное масло, Свинина, Репчатый лук, Чеснок, Имбирь => Жареная свинина по-китайски {Путем прожаривания свинины со сливочным маслом, репчатым луком, чесноком и имбирем получаем Жареную свинина по-китайски.}
Плита, Вареная лапша, Жареная свинина по-китайски, Кунжут => Лапша по-китайски! {Добавляем вареную лапшу и жареную свинину по-китайски. Сверху присыпаем кунжутом. Получаем Лапшу по-китайски.}
Манго, Ананас, Апельсин, Вода, Плита, Карамель => Фруктовое варенье {Путем перетирания манго, ананаса, апельсина с добавлением воды и карамели при нагревании образуется Фруктовое варенье. }
Вода, Холодильник => Лед {Вода, оставленная в холодильнике образует Лед.}
Холодильник, Фруктовое варенье, Лед  =>  Фруктовый лед { Путем смешения фруктового варенья и льда при воздействии низких температур получим Фруктовый лед.}
Фруктовый лед, Молоко => Фруктовое молоко! { Путем смешения фруктового льда и молока получим Фруктовое молоко.}
Тесто => Лист лазаньи { Из теста формируются листы лазаньи.} 
Тесто => Корж для пиццы { Из теста формируется корж для пиццы.} 
Помидор, Соль => Томатная паста { Из помидоров с добавлением соли получаем томатную пасту. } 
Печь, Лист лазаньи, Томатная паста, Моцарелла, Базилик => Лазанья! { Добавляем базилик к томатной пасте, получаем соус. Соус наносим на листья лазаньи, посыпаем Моцареллой, выпекаем в печи. } 
Печь, Корж для пиццы, Томатная паста, Томаты черри, Моцарелла, Базилик => Пицца Маргарита { На корж наносим томатную пасту, посыпаем моцареллой, выкладываем базилик и томаты черри, выпекаем в печи.} 
Говядина, Свинина, Соль, Перец => Пепперони { Смешиваем говядину и свинину, добавляем соль и перец, получаем Пепперони. } 
Печь, Корж для пиццы, Томатная паста, Моцарелла, Пепперони => Пицца Пепперони { На корж наносим томатную пасту, посыпаем моцареллой, выкладываем Пепперони, выпекаем в печи} 
Плита, Вода, Курица, Соль, Перец => Куриный бульон { Варим курицу в воде, добавив соль и перец. } 
Плита, Рис, Сливочное масло, Куриный бульон => Ризотто { Варим рис со сливочным маслом, добавляем куриный бульон, получаем ризотто. }
Молоко, Холодильник => Охлаждённое молоко {Под воздействием низких температур в холодильнике получаем охлаждённое молоко. }
Кефирные грибки, Охлаждённое молоко => Закваска {Путём добавления в пастеризованное и охлажденное молоко добавляем кефирные грибки. Выдерживаем и получаем Закваску. }
Плита, Молоко, Вода, Бактерии => Закваска {В разогретую в кастрюле с водой банку с молоком добавляем Бактерии. Под воздействием комфортных для бактерий температур получим Закваску. }
Молоко, Закваска => Кефир {С добавлением в молока закваску и выдержкой определенного времени получим кефир. }
Молоко, Закваска => Сыр {С добавлением в молока закваску и выдержкой опреденного времени получим Сыр. }
Кефир, Сахар, Соль, Дрожжи, Яйцо, Подсолнечное масло, Мука => Тесто {Смешав в одной посуде Кефир, Сахар, Соль, Дрожжи, Яйцо, Подсолнечное масло и Муку получим Тесто. } 
Духовка, Тесто, Сыр, Сливочное масло => Хачапури! {Запекаем Тесто с добавлением Сыра в качестве начинки. Получаем Хачапури. }
Мука, Соль, Яйцо, Вода => Тесто { Смешав в одной посуде Муку, Соль, Яйцо и Воду получим Тесто. }
Говядина, Сало, Лук, Кинза, Перец красный копчёный сухой, Перец чёрный молотый, Перец острый стручковый => Фарш {Пропустив через мясорубку Говидину с Салом получим фарш. } 
Плита, Вода, Соль, Фарш, Тесто => Хинкаль! { Фарш заворачиваем в Тесто. Получившиеся заготовки отправляем в подсоленную Воду, получаем Хинкаль. }
Вода, Подсолнечное масло, Соль, Сахар, Дрожжи, Мука => Тесто {Смешав Воду, Подсолнечное масло, Соль, Сахар, Дрожжи и Муку получим Тесто. }
Тесто, Печь => Хлеб {При выпекании теста получим Хлеб. }
Хлеб, Духовка => Гренки {Нарезав хлеб, отправляем в духовку. Под высокими температурами получим гренки. }
Лук репчатый, Сахар, Сливочное масло => Карамелизированный лук лук {Обжариваем Лук репчатый, постепенно добавляем Сахар и Сливочное масло. Получаем Карамелизированный лук. }
Куриный бульон, Карамелизированный лук, Сыр, Гренки, Духовка => Луковый суп! {Добавляем в Карамелизированный лук Куриный бульон. Посыпаем Сыром и отправляем в Духовку. Сверху кладём Гренки и получаем Луковый суп. }
Мангал, Курдючное сало, Подсолнечное масло => Шкварки {Жарим на Подсолнечном масле сало. Дожидаемся, когда оно вытопится и получаем Шкварки. }
Мангал, Говядина, Масло => Жаренная говядина {Жарим Говядину на Масле и получаем Жаренную говядину. }
Мангал, Жаренная говядина, Лук репчатый, Морковь, Перец острый стручковый, Вода, Зира молотая, Соль, Перец чёрный молотый => Тушёная говядина { Добавив к Жаренной Говядине Лук репчатый, Морковь, Перец острый стручковый, Воду, Зира молотая, Соль, Перец чёрный молотый и оставив тушиться мы получим Тушёную говядину. }
Мангал, Тушёная говядина, Шкварки, Рис отварной, Вода => Плов { Смешав Тушёную говядину с Рисом и Шкварками, добавив немного воды и оставив до её выпаривания, получим плов. }
